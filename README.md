# 🍽️ API REST - Gestión de Reservaciones de Restaurante

API REST profesional desarrollada con **Node.js**, **Express** y **Prisma ORM**, utilizando autenticación JWT, encriptación de contraseñas con bcrypt, control de acceso por roles y persistencia en una base de datos PostgreSQL.

---

## 🚀 Requisitos Previos

Antes de instalar el proyecto, asegúrate de tener instalado:
* **Node.js** (v18 o superior)
* **PostgreSQL** (Base de datos ejecutándose)
* **Git**

---
